A new Flutter project.

## Getting Started

Para rodar no vscode criar no .launch.json o arquivo

````{

    "version": "0.2.0",
    "configurations": [
        {
            "name": "PROD",
            "request": "launch",
            "type": "dart"
        },
        {
            "name": "DEV",
            "request": "launch",
            "type": "dart",
            "program": "lib/main_dev.dart",
        },

    ]
}```
````
